# 会员等级规则

## 等级划分
普通会员：注册即可，享受 7 天无理由退货。
银卡会员：年消费满 2000 元，享受 15 天无理由退货 + 优先客服。
金卡会员：年消费满 5000 元，享受 30 天无理由退货 + 专属客服 + 免运费退货。
钻石会员：年消费满 10000 元，享受 45 天无理由退货 + 专属客服 + 免运费退货 + 极速退款。

## 退款处理优先级
高等级会员的退款申请需要优先处理。钻石/金卡会员退款需在 2 小时内响应。

## 售后权益对照
| 权益 | 普通 | 银卡 | 金卡 | 钻石 |
| 退货期限 | 7天 | 15天 | 30天 | 45天 |
| 运费补贴 | 无 | 无 | 免退货运费 | 免退货运费 |
| 极速退款 | 否 | 否 | 否 | 是 |
